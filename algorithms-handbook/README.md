# Algorithms Handbook

A comprehensive, practical guide to algorithms and data structures for CS students and engineers. All code examples are in **Java**. Designed to be read linearly as a complete course or consulted per-module as a reference.

---

## Conventions

- **Language:** Java 17+
- **Math:** LaTeX notation ($O(n \log n)$, $\Theta(n^2)$)
- **Each module** ends with a quiz (5--8 questions with answers)
- **Target audience:** CS students and software engineers
