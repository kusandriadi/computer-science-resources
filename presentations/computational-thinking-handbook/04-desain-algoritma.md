# Desain Algoritma (Algorithm Design) 📋
### Membuat Instruksi Langkah demi Langkah!

> Resep masakan, petunjuk arah, aturan permainan — semua itu algoritma!

---

# Apa yang Akan Kita Pelajari

- Apa itu **Algoritma** (Algorithm)
- Bahan-bahan algoritma yang bagus
- Algoritma dengan keputusan (JIKA-MAKA)
- Algoritma dengan pengulangan (LOOP)
- Aktivitas seru: Permainan Robot! 🤖

---

# Apa Itu Algoritma? 🤔

> **Algoritma** = kumpulan instruksi langkah demi langkah untuk menyelesaikan masalah

Tiga hal penting:
1. Punya **langkah-langkah yang jelas** ✅
2. Langkahnya dalam **urutan tertentu** ✅
3. Siapa pun yang ikuti → dapat **hasil yang sama** ✅

> Kamu ikuti algoritma setiap hari tanpa sadar!

---

# Contoh: Resep Roti Isi Cokelat 🍫

```
Langkah 1: Ambil dua lembar roti
Langkah 2: Ambil toples selai cokelat dan mentega
Langkah 3: Ambil pisau roti
Langkah 4: Buka toples selai cokelat
Langkah 5: Oleskan selai di satu lembar roti
Langkah 6: Oleskan mentega di lembar lain
Langkah 7: Tempelkan kedua lembar roti
Langkah 8: Potong jadi dua
Langkah 9: Makan dan nikmati! 😋
```

> Siapa pun ikuti langkah ini → dapat roti isi enak!

---

# Contoh: Aturan Kejar-kejaran 🏃

```
Langkah 1: Pilih satu orang jadi "pengejar"
Langkah 2: Semua yang lain berlari menjauh
Langkah 3: Si pengejar mengejar yang lainnya
Langkah 4: Kalau pengejar menyentuh seseorang
           → orang itu jadi pengejar baru
Langkah 5: Pengejar baru hitung sampai 5
Langkah 6: Ulangi dari Langkah 3!
```

> Aturan permainan = algoritma!

---

# Bahan Algoritma yang Bagus ⭐

### 1. Langkah yang Jelas
- ❌ "Masak makanannya"
- ✅ "Masukkan 2 sendok beras ke panci, tambah air"

### 2. Urutan yang Benar
- ❌ Makan kue → Panggang kue (terbalik!)

### 3. Nggak Ada yang Terlewat
- ❌ Cuci tangan tanpa sabun!

### 4. Awal dan Akhir yang Jelas
- Harus tahu kapan mulai dan kapan selesai!

---

# Algoritma dengan Keputusan: JIKA-MAKA 🔀

Kadang ada titik keputusan — harus pilih!

```
Langkah 1: Cek cuaca
Langkah 2: JIKA cerah → pakai kaos dan celana pendek ☀️
           JIKA hujan → pakai jaket dan bawa payung 🌧️
           JIKA dingin → pakai sweater 🧥
Langkah 3: Pakai kaus kaki dan sepatu
Langkah 4: Lihat cermin — selesai! 🪞
```

> JIKA-MAKA = persimpangan jalan — kamu pilih satu arah!

---

# Algoritma dengan Pengulangan: LOOP 🔄

Kadang satu langkah perlu diulang terus-menerus!

**Contoh: Makan Sereal 🥣**
```
Langkah 1: Tuang sereal ke mangkuk
Langkah 2: Tuang susu
Langkah 3: Ambil sendok
Langkah 4: ULANGI SAMPAI mangkuk kosong:
             → Sendok sereal
             → Masukkan ke mulut
             → Kunyah dan telan
Langkah 5: Taruh mangkuk di bak cuci
```

> Nggak perlu tulis "suapan 1, suapan 2, suapan 3..." — cukup bilang "ulangi!"

---

# Tanpa Algoritma vs. Dengan Algoritma 📊

| Situasi | Tanpa Algoritma ❌ | Dengan Algoritma ✅ |
|---|---|---|
| Memanggang | "Masukin aja ke oven!" → gosong 🔥 | Ikuti resep → kue enak! 🎂 |
| Petunjuk arah | "Jalan ke sana agak jauh" → tersesat 😭 | "2 blok, belok kiri" → sampai! 🎯 |
| Main game | "Tinggal pukul bola!" → ribut 😤 | Aturan jelas → seru! 🎉 |

# Pseudocode: Algoritma Bahasa Sehari-hari 💬

```
MULAI
  Cek tas untuk PR
  JIKA ada PR MAKA
    Kerjakan PR dulu
    KALAU sudah selesai → pergi main
  KALAU TIDAK
    Langsung pergi main!
  SELESAI JIKA
  JIKA sudah gelap MAKA
    Pulang ke rumah
  SELESAI JIKA
  Makan malam → Sikat gigi → Tidur 😴
SELESAI
```

> Cuma bahasa sehari-hari dengan sedikit struktur!

# Aktivitas: Permainan Robot 🤖

**Cara bermain:**
1. Taruh "harta karun" di suatu tempat di ruangan
2. Satu orang jadi **Robot**, satu jadi **Programmer**
3. Robot HANYA bisa:
   - "Langkah maju" | "Langkah mundur"
   - "Belok kiri" | "Belok kanan"
   - "Ambil" | "Taruh"
4. Programmer memandu Robot ke harta karun!

**Aturan:** Robot ikuti PERSIS — kalau bilang "maju" dan ada tembok, Robot nabrak! 💥

# Ingat! 🌟

- **Algoritma** = instruksi langkah demi langkah
- Algoritma bagus: **jelas**, **urut**, **lengkap**, ada **awal dan akhir**
- Bisa tambah **keputusan** (JIKA-MAKA)
- Bisa tambah **pengulangan** (ULANGI SAMPAI)
- Resep, petunjuk arah, aturan game — semua algoritma!

> **Kalau kamu bisa tulis langkah-langkah jelas, kamu bisa desain algoritma!** 📋
