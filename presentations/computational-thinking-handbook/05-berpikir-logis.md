# Berpikir Logis (Logical Thinking) 🧠
### Membuat Keputusan dengan JIKA/MAKA, Benar/Salah!

> Setiap hari kamu membuat ratusan keputusan — yuk bikin lebih CERDAS!

---

# Apa yang Akan Kita Pelajari

- Apa itu **Berpikir Logis** (Logical Thinking)
- Pernyataan **JIKA-MAKA** dan **SELAINNYA**
- Operator **DAN**, **ATAU**, **BUKAN**
- **Pohon Keputusan** (Decision Tree)
- Aktivitas seru: 20 Pertanyaan dan Teka-Teki! 🧩

---

# Apa Itu Berpikir Logis? 🤔

> **Berpikir Logis** = menggunakan aturan, fakta, dan penalaran untuk membuat keputusan

Alat-alat keren berpikir logis:

| Alat | Contoh |
|---|---|
| **Ya / Tidak** | "Apakah sedang hujan?" |
| **Benar / Salah** | "Kucing bisa terbang" → SALAH! |
| **JIKA / MAKA** | "JIKA hujan, MAKA bawa payung" |
| **DAN, ATAU, BUKAN** | Menggabungkan syarat-syarat |

---

# Berpikir JIKA-MAKA 🔀

> **JIKA** sesuatu benar, **MAKA** lakukan sesuatu

Kamu pakai ini SEPANJANG WAKTU:

- JIKA lapar → MAKA makan camilan 🍪
- JIKA lampu merah → MAKA berhenti 🚦
- JIKA teman lambaikan tangan → MAKA lambaikan balik 👋
- JIKA hari Sabtu → MAKA nggak ke sekolah 🎉
- JIKA baterai HP habis → MAKA charge! 🔋

---

# JIKA-MAKA-SELAINNYA 🔀

Kadang ada juga **SELAINNYA** — apa yang dilakukan kalau TIDAK terpenuhi:

```
JIKA hujan
  MAKA bawa payung ☔
SELAINNYA
  Pakai kacamata hitam 😎

JIKA ada PR
  MAKA kerjakan PR dulu 📚
SELAINNYA
  Langsung main! 🎮

JIKA tokonya buka
  MAKA pergi beli susu 🥛
SELAINNYA
  Coba lagi besok
```

---

# Contoh: Lampu Lalu Lintas 🚦

Lampu lalu lintas = sistem logika sederhana!

| Lampu | Aksi |
|---|---|
| 🟢 **Hijau** | MAKA jalan! |
| 🟡 **Kuning** | MAKA pelankan dan bersiap berhenti |
| 🔴 **Merah** | MAKA berhenti! |

> Semua orang ikuti logika yang sama → semua aman! 🛡️

---

# DAN, ATAU, BUKAN 🔗

### DAN — Keduanya harus benar
JIKA hari Sabtu **DAN** cuaca bagus → pergi ke taman
(Sabtu + hujan? Nggak jadi!) 😅

### ATAU — Setidaknya satu benar
JIKA punya pensil **ATAU** pulpen → bisa menulis
(Cuma butuh salah satu!) ✏️

### BUKAN — Kebalikannya
JIKA **BUKAN** hujan → main di luar!
(Tidak hujan = bisa main!) ☀️

---

# Contoh Gabungan DAN, ATAU, BUKAN 🧩

**Sederhana:**
> JIKA cerah **DAN** ada waktu luang → main di luar! 🌞

**Lebih kompleks:**
> JIKA hari ulang tahunku
> **DAN** **BUKAN** hujan
> **DAN** (teman bisa datang **ATAU** sepupu bisa datang)
> → Adakan pesta di luar! 🎂🎉

Pecah satu-satu → setiap bagian masuk akal!

---

# Benar atau Salah? ✅❌

| Pernyataan | Jawaban |
|---|---|
| Matahari itu panas ☀️ | ✅ BENAR |
| Ikan bisa jalan 🐟 | ❌ SALAH |
| 2 + 3 = 5 | ✅ BENAR |
| Pisang berwarna biru 🍌 | ❌ SALAH |
| Semua burung bisa terbang 🐧 | ❌ SALAH (penguin!) |
| Semua persegi adalah persegi panjang | ✅ BENAR |

> Berpikir logis = periksa dengan cermat, jangan cuma menebak!

---

# Pohon Keputusan (Decision Tree) 🌳

```
         Apakah kamu lapar?
           /          \
         YA           TIDAK
         /              \
   Makan camilan    Apakah ada PR?
                     /        \
                   YA         TIDAK
                   /            \
            Kerjakan PR    Ada teman yg bisa main?
                             /        \
                           YA         TIDAK
                           /            \
                   Main bersama!   Baca buku 📖
```

> Mulai dari atas, ikuti cabang berdasarkan jawabanmu!

# Aktivitas: Permainan 20 Pertanyaan 🎯

**Cara bermain:**
1. Pemain 1 pikirkan sebuah benda
2. Pemain lain tanya sampai **20 pertanyaan Ya/Tidak**
3. Nggak boleh tanya "apa" atau "bagaimana"!

**Tips strategi — tanya yang LUAS dulu:**
- "Apakah benda itu hidup?" (singkirkan setengah kemungkinan!)
- "Apakah lebih besar dari kotak roti?"
- "Apakah bisa dimakan?"

> Pertanyaan luas → singkirkan BANYAK pilihan sekaligus!

---

# Aktivitas: Teka-Teki Logika 🧩

**Teka-teki: Siapa duduk di mana?**

Tiga teman: **Amir, Bella, Carlos** — tiga kursi: kiri, tengah, kanan

- Amir BUKAN di kiri
- Bella BUKAN di tengah
- Carlos di kanan

**Jawabannya:**
- Carlos → kanan ✅
- Amir bukan di kiri → Amir di **tengah** ✅
- Bella → **kiri** ✅

> Mulai dari yang pasti, lalu cari tahu sisanya!

# Kuis Singkat! 🧩

**1.** JIKA hari sekolah **DAN** aku sehat → pergi sekolah.
Hari ini Senin dan aku sakit. Pergi sekolah?
- **B) Tidak** ✅ (DAN butuh KEDUA syarat benar!)

**2.** JIKA punya uang **ATAU** kartu hadiah → bisa beli buku.
Nggak punya uang, tapi punya kartu hadiah. Bisa beli?
- **A) Ya** ✅ (ATAU cuma butuh SATU yang benar!)

# Ingat! 🌟

- **Berpikir Logis** = gunakan aturan dan penalaran untuk keputusan cerdas
- **JIKA-MAKA** = buat keputusan berdasarkan syarat
- **DAN** = kedua syarat harus benar
- **ATAU** = minimal satu harus benar
- **BUKAN** = kebalikannya
- **Pohon Keputusan** = peta visual untuk serangkaian pilihan

> **Kamu pakai berpikir logis setiap hari — sekarang kamu lebih JAGO!** 🧠
